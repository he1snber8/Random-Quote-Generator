# Bitasmbl-Random-Quote-Generator-045385-Bitasmbl-Bot

## Description
Build a web application that fetches and displays random quotes from an API. This project introduces API fetching, dynamic content rendering, and basic interactivity.

## Tech Stack
- GraphQL
- Vue.js

## Requirements
- Fetch quotes from a public API
- Display the quote dynamically on the page
- Allow users to get a new random quote on button click
- Ensure the application is responsive and visually clear

## Installation
bash
git clone https://github.com/he1snber8/Random-Quote-Generator.git
cd Random-Quote-Generator
npm install


## Usage
bash
npm run dev

Open the shown local URL in a browser.

## Implementation Steps
1. Initialize a Vue.js project.
2. Create a main Quote component.
3. Configure GraphQL client to query a public quote API.
4. Fetch a random quote on component mount.
5. Render quote text and author reactively.
6. Add a button to refetch a random quote.
7. Apply responsive styling.
